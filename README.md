# Star Seeker

React Native app for exploring gates, calculating transport costs, and finding routes.

## How to Run

```bash
npm install
npm start
```

Then press `i` for iOS, `a` for Android, or scan the QR code with Expo Go.

## How to Test

```bash
npm test
```

## How to Use

Three tabs at the bottom:

- **Gates**: Browse gates and view their connections
- **Transport**: Calculate transport costs
- **Route**: Pick start and end gates to see the route and cost
